## 🔧 Technical Details

### ESP8266 vs ESP32 TLS Differences

**ESP32** (uses `WiFi.h`):
```cpp
#include <WiFi.h>
#include <WiFiClientSecure.h>

WiFiClientSecure espClient;
espClient.setCACert(ca_cert);  // ESP32 method
```

**ESP8266** (uses `ESP8266WiFi.h`):
```cpp
#include <ESP8266WiFi.h>
#include <WiFiClientSecure.h>
#include <BearSSL.h>

WiFiClientSecure espClient;
BearSSL::X509List cert(ca_cert);  // Global variable!
espClient.setTrustAnchors(&cert);  // ESP8266/BearSSL method
```

**Critical Note**: The `X509List` object MUST be global, not local in `setup()`, otherwise it will be destroyed after setup exits.
